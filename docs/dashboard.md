# Dashboard Documentation

## Overview

The Streamlit dashboard provides an interactive UI for visualizing risk metrics, exploring model predictions, and chatting with the Groq-powered assistant.

## Running the Dashboard

```bash
python Scripts/matrisk_step5_dashboard.py
```

The app will launch at `http://localhost:8501`.

## Main Sections

1. **Data Overview** – Displays loaded datasets and summary statistics.
2. **Model Insights** – Shows feature importance charts and calibration plots.
3. **Forecasts** – Interactive time‑series plots generated by Prophet.
4. **Chat Assistant** – A chatbot powered by Groq that can answer questions about the pipeline, data, or results.

## Customising the Dashboard

- Edit `Scripts/matrisk_step5_dashboard.py` to change the layout or add new pages.
- Modify the environment variable `GROQ_API_KEY` in your `.env` to change the API key.
- Adjust the Streamlit config (`.streamlit/config.toml`) for theme or server settings.
