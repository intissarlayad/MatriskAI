## À propos

**MatriskAI** est un pipeline complet de Data Science & Machine Learning en **4 étapes** conçu pour évaluer, prédire et prescrire des actions sur les risques liés aux matériaux industriels (QML/ASL), en combinant :

- 🔧 **Feature Engineering** avancé (13 features extraites de 50 colonnes brutes)
- 🧠 **XGBoost** calibré + détection d'anomalies (IsolationForest)
- 📈 **Prophet** (séries temporelles bayésiennes) pour les prévisions J+90
- ⚡ **Moteur Prescriptif** (15 règles métier → 653 actions)
- 🖥️ **Dashboard Streamlit** interactif avec assistant IA (Groq)
